# Public asset guide

All landing, brand and product images live under this folder so you always know where to drop files.

```
public/
└─ assets/
   ├─ brand/               # Logos, favicons, color variations
   ├─ landing/
   │   ├─ hero/            # Backgrounds, portraits or decorative hero art
   │   ├─ mockups/         # UI mockups used in the "Producto" section
   │   └─ sections/
   │       ├─ highlights/  # Icons or feature illustrations
   │       ├─ how-it-works/# Step-by-step illustrations
   │       └─ cta/         # Images for the final call-to-action strip
   └─ product/
       ├─ dashboard/       # Desktop screenshots of FitBalance
       └─ mobile/          # Mobile/PWA previews
```

> Tip: any file you place here is available publicly at `/assets/...` (for example `public/assets/brand/logo.svg` is served at `https://yourdomain.com/assets/brand/logo.svg`).
