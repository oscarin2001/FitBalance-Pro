# Brand assets

Drop every FitBalance logo, favicon or color variation here. Recommended filenames:

- `logo-full.svg` — main horizontal logo
- `logo-mark.svg` — icon-only version
- `logo-dark.png` / `logo-light.png` — raster exports for social previews
- `favicon-*.png` — 32x32, 64x64 etc.

Once a file is placed here you can reference it in the app using `/assets/brand/<file-name>`.
